Para ejecutar:

-Descomprimir el proyecto en la carpeta del disco C:. La ruta del proyecto una vez descomprimido debe ser C:\proyecto
-Dentro de la carpeta "proyecto", dirigirse a la siguiente ruta: C:\proyecto\httpd-2.4.28-o102l-x86-vc14\Apache24\bin
y ejecutar httpd.exe. Si el sistema operativo solicita permisos de Firewall, otorgarlos.
-Finalmente, con httpd.exe ejecutándose, abrir algún navegador web de su preferencia y dirigirse al siguiente enlace:
http://localhost:8085/seleccion_primeraEtapa

Para revisar el código:

-Dentro de la carpeta "proyecto" dirigirse a la siguiente ruta: C:\proyecto\httpd-2.4.28-o102l-x86-vc14\Apache24\htdocs
-El nombre del proyecto es seleccion_primeraEtapa
