Para ejecutar:

-Clonar el proyecto en la carpeta del disco C:. La ruta del proyecto una vez descomprimido debe ser:
C:\seleccion_PrimeraEtapa
-Dentro de la carpeta "seleccion_PrimeraEtapa", dirigirse a la siguiente ruta:
C:\seleccion_PrimeraEtapa\httpd-2.4.28-o102l-x86-vc14\Apache24\bin y ejecutar httpd.exe. Si el sistema operativo solicita
permisos de Firewall, otorgarlos
-Finalmente, con httpd.exe ejecutándose, abrir algún navegador web de su preferencia y dirigirse al siguiente enlace:
http://localhost:8085/seleccion_primeraEtapa

Para revisar el código:

-Dentro de la carpeta "seleccion_PrimeraEtapa" dirigirse a la siguiente ruta:
C:\seleccion_PrimeraEtapa\httpd-2.4.28-o102l-x86-vc14\Apache24\htdocs
-El nombre de la carpeta contenedora del proyecto es seleccion_primeraEtapa

Observación:

-Se recomienda ejecutar el programa en un entorno windows, de lo contrario se deberán instalar y configurar PHP y Apache
en base a los requerimientos del SO utilizado
